# Day 2: SQL Practice – SELECT, FROM, WHERE 

### ✅ What I Practiced Today:
On Day 2 of my SQL learning journey, I focused on the basic but most important:
- `SELECT` – to choose specific columns from a table
- `FROM` – to specify the table to retrieve data from
- `WHERE` – to filter records based on specific conditions
---

 ### 🧠 Concepts Covered:
- Selecting all columns using `SELECT *`
- Selecting specific columns like `name`, `age`, `salary`
- Filtering rows using conditions:
  - Numeric comparisons (`>`, `<`, `=`, `!=`)
  - Logical operators (`AND`, `OR`, `NOT`)
  - Date filtering with `hire_date`
  - Null check using `IS NULL`
