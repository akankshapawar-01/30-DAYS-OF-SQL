## 30 Days of SQL – Learn SQL with Real-World HR Dataset
Daily SQL practice for 30 days to build strong database skills. Covers fundamentals, joins, subqueries, functions, and real-world query problems.

## 🔧 Getting Started – Setup Instructions
To practice the queries, install and set up **MySQL Workbench** and **MySQL Server**:

### 🖥️ Install MySQL (Windows/Mac/Linux)
1. Download and install MySQL from: https://dev.mysql.com/downloads/installer/
2. During setup, install:
   - MySQL Server
   - MySQL Workbench
3. Set a root password during installation
4. Launch **MySQL Workbench** and connect to your local server
---

## 🗂️ Project Setup
### Step 1: Create a Schema
CREATE DATABASE employee_db;
USE employee_db;

### Step 2: Run SQL Scripts
- Open and run create_tables.sql to create tables
- Open and run insert_data.sql to insert sample records

### 🎯 Objective
- Build a strong foundation in SQL using a custom HR dataset

- Improve query writing, database design, and data analysis skills

### 🗃️ About the Dataset
- This project uses a self-created HR database consisting of two main tables:
- departments :– stores department names and IDs
- employees :– stores employee info including name, age, salary, and hire date

The schema and initial data are created on Day 0 using:
- create_tables.sql
- insert_data.sql
