## What is a Database ❓
- A **Database** is an organized collection of data, generally stored and accessed electronically from a computer system.  
---

## What is DBMS ❓
- A **Database Management System (DBMS)** is software that handles the storage, retrieval, and updating of data in a database.
---

## What is RDBMS ❓
- A **Relational Database Management System (RDBMS)** is a type of DBMS that stores data in tables with relationships between them.  
Examples: MySQL, PostgreSQL, Oracle, SQL Server  
---

## What is SQL ❓
SQL (Structured Query Language) is a standard language for accessing and manipulating databases.

🧠 Why SQL is Important ?
- Universally used to manage and query databases
- Core to backend development, data analysis, and business intelligence
- Efficient data manipulation, access control, and transaction handling

  📌Note: SQL statements are usually terminated with a semicolon (`;`). SQL keywords are not case-sensitive but are conventionally written in uppercase.
---

## 📊 Types of SQL Statements

| Category | Full Form | Purpose |
|----------|-----------|---------|
|    DDL   | Data Definition Language | Define database structure (`CREATE`, `DROP`, `ALTER`, `TRUNCATE`) |
|    DML   | Data Manipulation Language | Manipulate data (`INSERT`, `UPDATE`, `DELETE`) |
|    DCL   | Data Control Language | Manage user access (`GRANT`, `REVOKE`) | 
|    DQL   | Data Query Language | Retrieve data (`SELECT`) |
|    TCL   | Transaction Control Language | Handle transactions (`COMMIT`, `SAVEPOINT`, `ROLLBACK`) |

---
