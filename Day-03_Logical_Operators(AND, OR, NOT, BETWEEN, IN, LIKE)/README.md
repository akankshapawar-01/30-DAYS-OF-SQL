## 📘 Day 3 – Logical Operators (AND, OR, NOT, BETWEEN, IN, LIKE)

## ✅ What I Learned Today
Logical operators in SQL are used to combine or modify multiple conditions within a query, enabling more precise data filtering and retrieval
- **Note** : SQL queries related to each operator are written in `query.sql`.
---

## 📘 Concepts Covered – Day 3
🔹 Logical Operators
- **AND** – Returns records where both conditions are true.
- **OR** – Returns records if any one condition is true.
- **NOT** – Negates a condition to filter out unwanted data.
- **BETWEEN** – Checks if a value lies within a range (inclusive).
- **IN** – Checks if a value exists in a set of values.
- **LIKE** – Used for pattern matching in text using wildcards:
    - `%` Represents zero or more characters
  - `_` Represents a single character
